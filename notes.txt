<!DOCTYPE html> tells web browser to use latest html version

HTML Structures: Opening / Closing tags
e.g. <head> </head>

<html> 
  this is the root, which contains all html elements 
</html>

in root (<html>):
 - <head> : contains meta data - things not displayed but help website function (e.g. browser <title>, stylesheets)
 - <body> : contains <div>, <h1-3>, <p>, <span>
